# mini Shopping Web

### Project

- 개요 : HTML, CSS, JS로 버튼으로 List를 보여주는 쇼핑몰 웹을 제작하였습니다.
- 언어 : HTML, CSS, JavaScript
- 기간 : 8시간

### 주요 기능

1. Button 클릭 시 List 필터링
2. logo 클릭 시 All List 필터링
